# 📚 Bibliothèque Numérique - Java POO

## 🎯 Description
Système de gestion d'une bibliothèque numérique 
développé en Java avec les principes de la 
Programmation Orientée Objet (POO) et l'utilisation 
des interfaces.

## 🛠️ Technologies
- Java (JDK 17+)
- Programmation Orientée Objet
- Interfaces Java

## ✨ Fonctionnalités
- 📖 Gestion des documents (Livres, Magazines, Thèses)
- 👤 Gestion des membres (Étudiants, Professeurs)
- 🔄 Système d'emprunts et retours
- 🔍 Recherche multicritères
- ⭐ Évaluation et commentaires
- 📊 Statistiques et rapports

## 🔧 Interfaces utilisées
- `Document` - Contrat de base pour tous les documents
- `Empruntable` - Gestion des emprunts
- `Recherchable` - Système de recherche
- `Evaluable` - Notation et commentaires
- `Notifiable` - Système de notifications

## 🚀 Comment lancer
```bash
git clone https://github.com/tonusername/BibliothequeNumerique-Java-POO
cd BibliothequeNumerique-Java-POO
javac Main.java
java Main
```
